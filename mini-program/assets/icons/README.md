# 图标文件

请在此目录放置以下图标文件（建议尺寸 81x81px，png格式）：

## TabBar 图标

| 文件名 | 用途 |
|--------|------|
| home.png | 首页-未选中 |
| home-active.png | 首页-选中 |
| case.png | 案件-未选中 |
| case-active.png | 案件-选中 |
| work.png | 工作台-未选中 |
| work-active.png | 工作台-选中 |
| my.png | 我的-未选中 |
| my-active.png | 我的-选中 |

## 其他图标

| 文件名 | 用途 |
|--------|------|
| default-avatar.png | 默认头像 |
| empty.png | 空状态图片 |
| marker.png | 地图标记 |
| marker-red.png | 地图标记(红色) |
| marker-blue.png | 地图标记(蓝色) |

## 图标设计建议

- 尺寸：81x81px（实际显示 48x48px）
- 格式：PNG
- 颜色：
  - 未选中：#999999
  - 选中：#1890ff
- 风格：线性图标，简洁明了
